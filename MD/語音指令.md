## 喚醒詞與結束詞

| 動作 | 說法 | 效果 |
|------|------|------|
| 喚醒系統 | `哈囉 曼波` | 系統進入聆聽模式，準備接收下一個指令 |

> 說喚醒詞後，系統播放提示音，再說功能指令。

## 盲道導航

沿著地面盲道（導盲磚）行走，系統自動辨識盲道位置並語音引導方向。

| 動作 | 說法 | 效果 |
|------|------|------|
| 啟動 | `開始導航` | 進入盲道導航模式 |
| 啟動 | `盲道導航` | 進入盲道導航模式 |
| 啟動 | `幫我導航` | 進入盲道導航模式 |
| 停止 | `停止導航` | 退出盲道導航模式，回到待機 |
| 停止 | `結束導航` | 退出盲道導航模式，回到待機 |

### 導航過程中的語音提示

系統會自動播報以下提示（無需說話）：

| 提示語 | 含義 |
|--------|------|
| 「請繼續直行。」 | 盲道在正前方，繼續走 |
| 「請向左平移。」 | 身體往左平移，盲道偏右 |
| 「請向右平移。」 | 身體往右平移，盲道偏左 |
| 「請向左轉。」 | 盲道向左轉彎 |
| 「請向右轉。」 | 盲道向右轉彎 |
| 「前方有障礙物，注意避讓。」 | 前方偵測到障礙物 |

## 過馬路輔助

偵測斑馬線位置與紅綠燈狀態，引導使用者安全過馬路。

| 動作 | 說法 | 效果 |
|------|------|------|
| 啟動 | `開始過馬路` | 進入過馬路流程（尋找斑馬線 → 等綠燈 → 引導通過） |
| 啟動 | `幫我過馬路` | 進入過馬路流程 |
| 停止 | `過馬路結束` | 停止過馬路流程，回到待機 |
| 停止 | `結束過馬路` | 停止過馬路流程，回到待機 |
| 手動確認通過 | `立即通過` | 在等待紅綠燈狀態下，手動強制通過（不建議） |
| 手動確認通過 | `現在通過` | 同上 |
| 手動確認通過 | `繼續` | 同上 |

### 過馬路自動流程

```
1. 尋找斑馬線
2. 對齊斑馬線方向
3. 等待綠燈（持續偵測紅綠燈）
4. 語音提示「可以通過」
5. 引導通過馬路
6. 尋找下一段盲道
```

## 紅綠燈偵測

單獨啟動紅綠燈偵測模式，持續識別燈號並語音回報，不進行完整過馬路流程。

| 動作 | 說法 | 效果 |
|------|------|------|
| 啟動 | `檢測紅綠燈` | 進入紅綠燈偵測模式，持續語音回報燈號狀態 |
| 啟動 | `看紅綠燈` | 進入紅綠燈偵測模式 |
| 停止 | `停止檢測` | 退出紅綠燈偵測模式 |
| 停止 | `停止紅綠燈` | 退出紅綠燈偵測模式 |

## 物品搜尋

使用 YOLO 開放詞彙偵測，協助找到指定物品。

| 動作 | 說法 | 效果 |
|------|------|------|
| 啟動 | `找一下〔物品名稱〕` | 開始在畫面中搜尋指定物品 |
| 啟動 | `幫我找一下〔物品名稱〕` | 開始在畫面中搜尋指定物品 |
| 停止 | `找到了` | 停止搜尋，若之前在導航中則自動恢復導航 |
| 停止 | `拿到了` | 停止搜尋，若之前在導航中則自動恢復導航 |

**說法範例：**
- 「找一下鑰匙」
- 「幫我找一下手機」
- 「找一下水杯」
- 「幫我找一下背包」

> 系統會先將中文物品名稱透過 AI 翻譯為英文 YOLO 標籤，再進行偵測，支援大多數日常物品。

## 視覺查詢（場景描述）

請 AI 描述當前攝影機畫面中的內容或識別前方物體。

| 說法 | 效果 |
|------|------|
| `前面有什麼` | 描述前方場景 |
| `前方有什麼` | 描述前方場景 |
| `前面是什麼` | 識別前方物體 |
| `幫我看` | 描述眼前場景 |
| `幫我看下` | 描述眼前場景 |
| `幫我看一下` | 描述眼前場景 |
| `看看` | 識別畫面中的物體 |
| `識別一下` | 識別畫面中的物體 |
| `那是什麼` | 詢問前方物體是什麼 |
| `前方情況` | 描述前方整體環境 |
| `幫我描述` | 描述當前環境 |
| `描述一下` | 描述當前環境 |

> 系統優先使用 **Nemotron 12B VLM** 回答（每次查詢間隔至少 3 秒），若失敗則降級至 Gemini。
> 在導航模式中也可使用以上關鍵字觸發視覺查詢，不會中斷導航。

## 打斷 AI 說話

若 AI 正在語音播報，可用以下指令立即停止：

| 說法 | 效果 |
|------|------|
| `停下` | 立即停止 AI 語音播放 |
| `別說了` | 立即停止 AI 語音播放 |
| `停止` | 立即停止 AI 語音播放 |

## 一般對話

不符合以上任何功能關鍵字的語音，自動進入 **AI 對話模式**，可結合攝影機畫面進行自由問答。

| 說法範例 | 效果 |
|----------|------|
| 「現在幾點？」 | AI 語音回答（無法查詢即時時間，會告知） |
| 「這附近有什麼？」 | AI 結合攝影機畫面回答 |
| 「幫我想一想...」 | AI 自由回應 |

> 在導航或紅綠燈偵測模式中，只有包含特定關鍵字（如「幫我看」、「找一下」）的語音才會進入 AI 對話，以避免一般話語誤觸發功能。

## 指令辨識說明

- **語音辨識有容錯**：不需要逐字說出完全一致的指令，例如「幫我開始導航」或「我要盲道導航」也可能被識別。
- **關鍵字比對**：系統在辨識到的文字中尋找關鍵詞，只要包含關鍵詞即可觸發。
- **麥克風靈敏度**：若喚醒詞常接收不到，可到 `.env` 調整 `ASR_PCM_GAIN`（增益）或降低 `ASR_STANDBY_RMS_THRESH`（靜音門檻）。
- **聲紋保護**：啟用聲紋驗證後，只有錄製聲紋的本人聲音才能觸發喚醒詞，可前往 `http://IP:8081/speaker` 管理。
